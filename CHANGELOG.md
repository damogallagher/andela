### Added

- FastAPI guardrail scanner with Postgres-backed scan history.
- React dashboard with uploads, severity filtering, searchable and sortable findings, pagination, scan history, trend, comparison, and SARIF export.
- CLI pipeline scanner with severity threshold exits.
- SARIF API endpoint and GitHub Code Scanning upload workflow.
- Terraform AWS infrastructure and deployment workflows.
- Alembic migrations for schema evolution.
- Structured JSON request logging, request IDs, CORS configuration, DB-aware health checks, and bounded upload handling.
- Repository hygiene files including license, security policy, editor configuration, changelog, and pre-commit lint gate.
